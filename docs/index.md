## LLM-friendly copies

The published site also exposes Markdown-first copies for AI assistants and
other tools that prefer compact source text:

- [`llms.txt`](https://stevennevins.github.io/daml-tools/llms.txt) is a small
  table of contents with short descriptions.
- [`llms-full.txt`](https://stevennevins.github.io/daml-tools/llms-full.txt)
  bundles the consumer docs into one file.
- Each content page is emitted beside its HTML page as a `.md` file.

These files are generated by VitePress during `npm run build`; edit the source
Markdown pages, not the generated files under `.vitepress/dist`.
